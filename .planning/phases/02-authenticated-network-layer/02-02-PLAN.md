<objective>
Implement `M_errors.from_http_status(status, body)` in src/errors.lua per D-24, replacing the Phase-1 stub. This is the single source of truth for HTTP-status-to-German-error mapping that Phase 2 and every subsequent phase consumes; Phase 5 will extend it additively without changing the signature.

Wave 3's entry-integration wiring depends on this contract being stable.
